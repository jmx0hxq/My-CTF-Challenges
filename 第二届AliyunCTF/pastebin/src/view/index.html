Welcome to Pastebin

Pastebin is a website where you can store any text online. The idea behind the site is to make it more convenient for people to share large amounts of text online.

It is popular with users of IRC and other chat programs where paste code is needed to show large amounts of text.

You can create a new paste, view recent pastes created by others, search pastes, and more.

Hope you enjoyed the pastebin last year. Here is a hint for you: no admin bot at the backend this time.
